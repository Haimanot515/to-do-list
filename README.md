## About

**To-Do List** is a simple and interactive web application that allows users to **add, edit, delete, and mark tasks as complete**, with all data persisted via a **JSON Server** backend. This project is designed to demonstrate **frontend skills**, responsive design, and basic CRUD (Create, Read, Update, Delete) operations in a structured, real-world workflow.

It is also a practical demonstration for learning how frontend applications can interact with a lightweight backend API using JSON.

---

## Features

- Add, edit, and delete tasks  
- Mark tasks as complete or incomplete  
- Tasks stored and managed using JSON Server  
- Responsive design for desktop and mobile  
- Clean and simple user interface  
- Demonstrates frontend-to-backend data interaction
