## Project Overview

**To-Do List** is a simple project built with **HTML, CSS, and JSON server** to demonstrate how to manage tasks in a structured way. Users can **add, update, and delete tasks**, with all data stored and fetched from a JSON server. This project helps understand the basics of **CRUD operations** and frontend-backend interaction using a lightweight JSON-based API.

---

## Features

- **Add new tasks**  
- **Edit existing tasks**  
- **Delete tasks**  
- **Mark tasks as complete/incomplete**  
- **Data persistence using JSON server**  
- **Clean and responsive design**  
- **Fully frontend-driven project**
